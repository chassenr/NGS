This directory contains scripts to extract a sample by OTU table from silvangs output.
Silvangs output was not designed to be used on OTU level!
Only use this scripts for comparative purposes, i.e. to compare silvangs on OTU level with the results of a different bioinformatic workflow.

Usage:
Run readSIVLA1.R first.
SILVAsampleByOtu_v1.3.R uses output of readSILVA function.
